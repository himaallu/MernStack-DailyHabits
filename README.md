# Daily Habits Tracker - MERN Stack

A simple **Daily Habits Tracker** built with the **MERN stack** (MongoDB, Express, React, Node.js).  
This project allows users to **add, toggle, and delete daily habits** with a backend API and a React frontend.

---

## Features

- Add new habits
- Toggle habit completion for the day
- Delete habits
- Fully connected to **MongoDB Atlas** for persistence
- CORS-enabled backend for frontend communication
